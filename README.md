# Practical - 2: CSS Card Layout

This project is a simple web page that demonstrates a card layout using only **HTML and CSS**. It was created as part of Practical - 2: *Create following layout using CSS layout properties*.

The page displays **four cards** (HTML, CSS, JavaScript, Bootstrap), each with:

- A colored icon area at the top
- A title
- A short description
- A “Read More” button with hover effect

## Files

- `index.html` – main webpage with the card structure.
- `style.css` – styles for layout, colors, spacing, and hover effects.

## How to run

1. Clone or download this repository.
2. Open `index.html` in any web browser.

No external libraries or images are required.
